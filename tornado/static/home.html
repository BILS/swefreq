Swedish Frequency Database

The SweGen dataset contains whole-genome sequences for 1000 individuals sampled from the Swedish population. A detailed description of the project can be found here [add link to BioRXiv article later]. This server hosts the variant frequency data generated within the SweGen project. This data can be used as a reference for the genetics research community and clinical genetics laboratories. Individual positions in the genome can be viewed using either the ExAC Browser or the Data Beacon by clicking the links above.

To access the variant frequency file you need to register.
